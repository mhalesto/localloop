# App Screenshots Guide

## Required Screenshots for App Store

### iPhone 6.7" Display (iPhone 14 Pro Max, 15 Pro Max, etc.)
**Required Size**: 1290 × 2796 pixels
**Format**: PNG or JPEG
**Minimum**: 3 screenshots
**Maximum**: 10 screenshots

### iPhone 6.5" Display (iPhone 11 Pro Max, XS Max, etc.)
**Required Size**: 1242 × 2688 pixels or 1284 × 2778 pixels
**Format**: PNG or JPEG

## How to Capture Screenshots

### Option 1: Device Simulator
1. Run app in iOS Simulator (iPhone 15 Pro Max)
2. Navigate to each screen
3. Press `Cmd + S` to save screenshot
4. Resize if needed to match required dimensions

### Option 2: Physical Device
1. Use iPhone 14 Pro Max or 15 Pro Max
2. Navigate to each screen
3. Press Volume Up + Side Button
4. Screenshots saved to Photos
5. AirDrop to Mac for editing

### Option 3: Automated (using expo-screen-capture)
Screenshots are already captured programmatically in your app. Check the app's screenshot functionality.

## Editing Tips

1. **Add Text Overlays** (optional but recommended):
   - Highlight key features
   - Use clear, readable fonts
   - Keep text concise (2-5 words)

2. **Maintain Consistency**:
   - Use same device frame for all screenshots
   - Similar lighting/theme across screenshots
   - Consistent spacing and layout

3. **Remove Sensitive Data**:
   - Use dummy/test account
   - No real user information
   - No profanity or inappropriate content

4. **Tools for Editing**:
   - Figma (add frames, text overlays)
   - Photoshop
   - Sketch
   - App Screenshot Maker websites

## Upload to App Store Connect

1. Go to your app in App Store Connect
2. Click "Distribution" → "iOS App" → Version 1.0
3. Scroll to "Previews and Screenshots"
4. Select "iPhone 6.7" Display" tab
5. Drag and drop screenshots in order
6. Click "Save"

**Note**: The first 3 screenshots are used on the App Store listing page, so make them count!
