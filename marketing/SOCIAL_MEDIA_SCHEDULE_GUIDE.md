## Quick Setup (Buffer Example)

### 1. Connect Your Accounts

1. Go to https://buffer.com
2. Sign up or log in
3. Click "Connect Social Accounts"
4. Add:
   - Twitter: @LocalLoopApp
   - Facebook: LocalLoop Page
   - Instagram: @localloopapp (requires Facebook connection)
   - LinkedIn: LocalLoop Company Page

### 2. Set Your Posting Schedule

**Optimal Times (South African audience):**
- Monday-Friday: 7 AM, 12 PM, 6 PM
- Saturday: 9 AM, 3 PM
- Sunday: 10 AM, 5 PM

**In Buffer:**
1. Go to Settings ‚Üí Posting Schedule
2. Set timezone to "Africa/Johannesburg"
3. Add posting times for each day

### 3. Upload Content

1. Copy post text from `SOCIAL_MEDIA_POSTS.md`
2. Click "Create Post" in Buffer
3. Paste content
4. Add image/video
5. Select platforms
6. Choose "Schedule" and pick date/time

## Analytics Tracking

### Set Up Tracking Links

Create UTM parameters for each platform:

```
App Store Link:
https://apps.apple.com/app/localloop/idXXX?utm_source=twitter&utm_medium=social&utm_campaign=gold_launch

Play Store Link:
https://play.google.com/store/apps/details?id=com.localloop&utm_source=facebook&utm_medium=social&utm_campaign=gold_launch
```

Use different UTM codes for each platform:
- utm_source: twitter, facebook, instagram, linkedin, tiktok
- utm_medium: social
- utm_campaign: gold_launch

**Tool:** https://ga-dev-tools.google/campaign-url-builder/

### Track These Metrics Daily:

**Platform Native Analytics:**
- Impressions
- Reach
- Engagement rate
- Click-through rate
- Video views / completion rate

## Emergency Procedures

### If a Post Gets Negative Reaction:

1. **Don't delete** (looks suspicious)
2. **Acknowledge** concerns politely
3. **Explain** feature value
4. **Offer** to discuss privately

**Example:**
```
"Thanks for the feedback! We understand pricing concerns.

Gold is designed for power users who want GPT-4o AI features. We still offer free Basic and affordable Premium (R50/month) for everyone else.

Happy to chat more - DM us! üòä"
```

### If Feature Isn't Working:

1. **Acknowledge** issue immediately
2. **Pause** promotional posts about that feature
3. **Post update** when fixed

**Example:**
```
"We're aware some users are experiencing issues with [feature].

Our team is working on a fix right now. We'll update you within 24 hours.

Thank you for your patience! üôè"
```
